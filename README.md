# Ветеринарная клиника «Мурмяу» 

8-я лабораторная работа.

## Функционал
- Просмотр заявок
- Добавление заявки
- Удаление заявки
- Перенос заявки на другую дату
- Работа с датами в PostgreSQL
- Красивый интерфейс с фоном и уведомлениями

## Технологии
- FastAPI (асинхронный)
- PostgreSQL
- Docker
- Pydantic
- Jinja2 (фронтенд)
